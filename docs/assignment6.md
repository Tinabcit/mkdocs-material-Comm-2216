# Low-Fidelity Wireframes 📒

These low-fidelity (LowFi) wireframes represent the early design structure of the Toronto Cupcake website redesign. They focus on layout, navigation, and user flow rather than visual styling.

!!! info "Purpose of LowFi Wireframes"
    LowFi wireframes are used to quickly explore layout ideas and user flows before applying colors, branding, and detailed design elements.

!!! warning "Design Limitation"
    At this stage, the designs do not include final visuals, typography, or branding. The focus is strictly on usability, structure, and content organization.

---

## Homepage Layout

![Homepage Wireframe](img/lowfi-homepage.png)

The homepage includes a **top navigation bar**, a **hero banner**, and a **grid of featured cupcakes**.

- Navigation includes: Home, Cupcakes, Order, Corporate, Events, Allergens, Contact  
- Featured products are displayed in a grid layout  
- A “View All” button allows users to explore more products  

**UX Improvement:**
- Clear structure improves first impressions  
- Grid layout supports easy scanning  

---

## Track Order Page

![Track Order Wireframe](img/lowfi-track-order.png)

This page allows users to track their order using an order code.

- Input field for order number  
- “Find my order” button  
- Order status and summary display  

!!! info "UX Insight"
    Providing a dedicated tracking page improves user confidence and reduces the need for customer support.

**UX Improvement:**
- Simplifies order tracking  
- Provides clear feedback to users  

---

## Cupcakes Catalogue Page

![Cupcakes Page](img/lowfi-cupcakes.png)

The cupcakes page displays products organized by **box size** and **flavors**.

- Options: 6 pcs, 12 pcs, 24 pcs, bulk  
- Grid layout for all cupcake flavors  
- Pricing shown under each item  

!!! warning "Usability Issue Addressed"
    In the original website, product organization was unclear. This layout improves structure and scanning.

**UX Improvement:**
- Better organization of products  
- Easier comparison between options  

---

## Product Details Page

![Product Details](img/lowfi-product.png)

This page shows detailed information about a selected cupcake.

- Product image  
- Quantity selector  
- “Add to Order” button  
- Allergen information (e.g., nuts, gluten)  

!!! danger "Accessibility Concern"
    Missing or unclear allergen information can create serious risks for users with dietary restrictions.

**UX Improvement:**
- Highlights important safety information  
- Improves decision-making for users  

---

## Allergens Page

![Allergens Page](img/lowfi-allergens.png)

This page provides important allergy-related information.

- Notes about gluten-free options  
- Warnings about cross-contamination  

**UX Improvement:**
- Centralized allergen information  
- Builds user trust and transparency  

---

## Cart Page

![Cart Page](img/lowfi-cart.png)

The cart page displays selected items before checkout.

- Product name and quantity  
- Remove button  
- Total price and deposit information  
- “Place Order” button  

!!! warning "Checkout Friction"
    If cart information is unclear, users may abandon their purchase before completing checkout.

**UX Improvement:**
- Clear pricing and totals  
- Easy editing of items  

---

## Checkout Page

![Checkout Page](img/lowfi-checkout.png)

The checkout page collects user information and payment details.

- Name, email, phone number  
- Payment section  
- “Pay Deposit” button  

!!! danger "Critical UX Step"
    A confusing checkout process can result in lost sales and user frustration.

**UX Improvement:**
- Simplified form layout  
- Clear payment process  

---

## Order Confirmation Page

![Order Confirmation](img/lowfi-confirmation.png)

This page confirms that the order has been placed successfully.

- Order tracking number  
- Confirmation message  
- Email notification reminder  

!!! info "User Feedback"
    Providing confirmation and tracking details reassures users that their order was successful.

**UX Improvement:**
- Builds trust  
- Provides clear next steps  

---

## Summary of Design Improvements

The LowFi wireframes address major usability issues identified in the original website:

- Improved navigation structure  
- Better product organization  
- Clearer checkout flow  
- Increased visibility of important information  

Overall, these wireframes focus on creating a more **user-friendly, efficient, and trustworthy experience**.
