# COMP 2811 – UX/UI Course Summary Website ⭐

## Overview
This project is a UX/UI course summary website based on the Toronto Cupcake redesign. It was completed as part of COMP 2811 and shows the full UX/UI design process from research to final design.

The website is built using **MkDocs** and includes multiple sections that explain each stage of the project.

## 📂 Project Structure
The project is divided into the following sections:

- Assignment 1 – Overview of Project  
- Assignment 2 – Norman Heuristic & UX/UI Proposal  
- Assignment 3 – User Stories and Personas  
- Assignment 4 – UX Problem Statement and Scope  
- Assignment 5 – User Flows and Task Flows  
- Assignment 6 – Low-Fidelity Wireframes  
- Assignment 7 – High-Fidelity Designs  
- Assignment 8 – Usability Testing and Feedback  
- Assignment 9 – Final Reflection and Improvements  

---

## 🧠 What This Project Covers
- UX analysis using Norman’s heuristics  
- User personas and user stories  
- UX problem definition and scope  
- User flows and task flows  
- Wireframing (low-fi and high-fi)  
- Usability testing and improvements  
- Final reflection
